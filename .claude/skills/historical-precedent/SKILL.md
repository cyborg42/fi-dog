# 历史先例匹配方法论

系统性地搜索、评分和对比历史上类似事件，推导恢复概率和时间线。

## 第一步：筛选可比案例

从知识库和网络搜索中，按以下三个维度筛选候选先例：

| 维度 | 匹配标准 |
|------|---------|
| **跌幅幅度** | 与当前案例跌幅相差 ±15个百分点以内 |
| **事件类型** | 同类触发因素（AI颠覆恐慌 / 运营事故 / 财报暴雷 / 监管打击 / 管理层变动） |
| **基本面受损程度** | 同级别（None / Mild / Moderate / Severe） |

**目标**：找到 3-5 个可比案例，涵盖"恢复"和"未恢复"两类结果。

## 第二步：恢复模式分类

每个先例按实际走势归入以下四种模式：

| 模式 | 特征 | 典型时间线 |
|------|------|-----------|
| **V型反弹** | 快速触底后强力回升，6个月内收复大部分跌幅 | 3-6个月 |
| **渐进V型** | 缓慢但稳定地回升，每次财报推动一段修复 | 6-12个月 |
| **L型横盘** | 触底后长期横盘，估值中枢下移但业务持续 | 12-24个月 |
| **持续下跌** | 基本面持续恶化，无恢复 | 无 |

**恢复 vs 不恢复的关键区分因素**：

| 恢复（V型/渐进V型） | 不恢复（L型/持续下跌） |
|---------------------|----------------------|
| AI威胁与核心产品无直接替代关系 | AI工具直接替代核心产品 |
| 管理层未确认业务受损 | 管理层承认AI影响 |
| 营收增速维持 | 营收增速放缓或转负 |
| 产品有不可替代的差异化壁垒 | 产品缺乏护城河 |
| 后续财报证明业务完好 | 后续财报证实恶化 |

## 第三步：可比性评分

对每个先例在以下维度打分（1-10），加权求和：

| 维度 | 权重 | 评分标准 |
|------|------|---------|
| 跌幅相似度 | 20% | 跌幅差距每5pct扣1分 |
| 事件类型匹配 | 25% | 完全相同10分，同大类8分，不同类5分 |
| 基本面受损程度匹配 | 25% | 同级别10分，相邻级别7分，差两级4分 |
| 公司质量可比性 | 15% | 增速、利润率、市场地位的相似度 |
| 时间接近度 | 15% | 近3年10分，3-5年8分，5-10年6分 |

**可比性评分 = Σ(维度分 × 权重)**

## 第四步：加权恢复概率

```
加权恢复概率 = Σ(先例恢复概率 × 可比性评分) / Σ(可比性评分)
```

先例恢复概率取值：
- V型反弹的先例：100%
- 渐进V型：100%
- L型横盘：30%（部分恢复）
- 持续下跌：0%

```
加权恢复时间 = Σ(先例恢复时间 × 可比性评分) / Σ(可比性评分)
```

## 已知案例参考库

随着分析积累持续更新：

### AI颠覆恐慌类
| 案例 | 跌幅 | 基本面受损 | 恢复模式 | 恢复时间 |
|------|------|-----------|---------|---------|
| Chegg 2023.05 (ChatGPT) | -48%→-99% | 严重 | 持续下跌 | 无 |
| Coursera 2023 (AI教育) | -12%→持续阴跌 | 轻微 | L型 | 无 |
| Adobe 2022-23 (AI创意工具) | -57%累计 | 无 | L型（估值压缩） | 无 |
| NET 2026.02 (Claude Code Security) | -38% | 无 | 待观察 | 待观察 |

### 运营事故/突发事件类
| 案例 | 跌幅 | 基本面受损 | 恢复模式 | 恢复时间 |
|------|------|-----------|---------|---------|
| CrowdStrike 2024.07 (全球IT宕机) | -36% | 无 | V型反弹 | 5-7个月 |
| PANW 2024.02 (平台化转型) | -25% | 无 | 渐进V型 | 6-10个月 |

### 关键发现
- **基本面未受损 + 威胁不存在直接替代**：恢复概率 >80%，典型时间 3-9个月
- **基本面未受损 + 但存在真实长期威胁**：L型概率高，估值长期承压
- **基本面实际受损**：恢复概率极低
