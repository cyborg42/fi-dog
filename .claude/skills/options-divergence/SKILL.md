# 期权市场背离检测框架

检测期权市场"聪明钱"与股票市场散户情绪之间的背离信号。

## 背离信号触发条件

当以下条件**同时满足**时，判定为"期权-股票背离（看多型）"：

1. 股票层面处于 SEVERE 或 EXTREME 冲击（单日跌幅 >-8%）
2. PCR（成交量）< 1.0（看涨成交量超过看跌）
3. 看涨/看跌成交比 > 1.5:1

满足2条为"弱背离信号"，满足全部3条为"强背离信号"。

## IV 结构分析

### 绝对水平

对比当前 IV 与该股票的历史区间：

| IV 位置 | 含义 |
|---------|------|
| >历史90分位 | 极端恐慌定价 |
| 70-90分位 | 高度不确定 |
| 30-70分位 | 正常范围 |
| <30分位 | 市场极度自满 |

若缺少历史分位数据，使用绝对值参考（高增长科技股）：

| IV 绝对值 | 含义 |
|-----------|------|
| >130% | 极端 |
| 90-130% | 偏高 |
| 50-90% | 正常 |
| <50% | 偏低 |

### 偏斜分析（Put-Call IV Spread）

| Put IV - Call IV | 含义 |
|-----------------|------|
| >20pct | 极端看跌偏斜，恐慌性保护需求 |
| 10-20pct | 显著偏斜，市场偏向保护 |
| 5-10pct | 轻微偏斜，正常范围 |
| <5pct | 几乎对称，市场未对下行收取额外溢价 |

**关键信号**：当股价暴跌但 IV 偏斜 < 5pct 时，说明期权做市商不认为下行风险需要额外溢价，这与恐慌叙事矛盾。

### 期限结构

如果能获取不同到期日的 IV：
- **短期 IV >> 长期 IV**（倒挂）：市场认为冲击是短暂的
- **短期 IV ≈ 长期 IV**（平坦）：市场认为不确定性会持续
- **短期 IV << 长期 IV**（正常）：事件已被消化

## 聪明钱方向判定

| 信号组合 | 判定 | 分析含义 |
|---------|------|---------|
| PCR<0.7 + 看涨2:1领先 + IV对称 | **强烈看多** | 机构积极抄底，恢复概率显著提升 |
| PCR 0.7-1.0 + IV对称 | **偏多** | 机构不认同恐慌，但尚未大举入场 |
| PCR 1.0-1.3 + IV轻微偏斜 | **中性** | 期权市场与股票市场一致 |
| PCR 1.3-1.5 + IV显著偏斜 | **偏空** | 机构在加大保护，恐慌有基础 |
| PCR>1.5 + IV极端偏斜 | **强烈看空** | 全面恐慌，可能接近投降底部 |

## 对恢复概率的影响

| 期权信号 | 恢复概率调整 |
|---------|-------------|
| 强烈看多 | +15% |
| 偏多 | +10% |
| 中性 | +0% |
| 偏空 | -10% |
| 强烈看空（但可能是投降底部） | 视RSI和成交量综合判断 |

## 输出格式

```
期权背离检测结果：
- 背离类型：强背离/弱背离/无背离
- 聪明钱方向：强烈看多/偏多/中性/偏空/强烈看空
- IV结构：对称/轻微偏斜/显著偏斜/极端偏斜
- 恢复概率调整：+X%
- 情绪极端性评分：X/10
```
