# 抛售归因分解框架

将股价跌幅拆解为不同驱动因素的贡献，并匹配适用的市场行为元模式。

## 第一步：四因素归因分解

### 因素 A：Beta 机械放大

高 Beta 股票在任何市场情绪冲击中都会被系统性放大。

```
Beta 调整后跌幅 = 实际跌幅 / Beta
Beta 放大贡献 = 实际跌幅 - Beta调整后跌幅
Beta 放大占比 = Beta放大贡献 / 实际跌幅
```

| Beta | 放大系数 | 说明 |
|------|---------|------|
| <1.2 | 低 | 跌幅基本反映真实冲击 |
| 1.2-1.5 | 中 | 约20-33%为放大效应 |
| 1.5-2.0 | 高 | 约33-50%为放大效应 |
| >2.0 | 极高 | 超过50%可能是机械放大 |

### 因素 B：ETF/被动卖出

当公司被纳入行业ETF，ETF赎回会产生与基本面无关的被动卖压。

**识别方法**：
- 成交量放大但 <2.5x（有序卖出而非恐慌投降）
- 同行业多只股票同步下跌，且跌幅与个股基本面关联度低
- 期权市场未确认恐慌（PCR 正常）

**估算方法**：
- 如果同行平均跌幅为 X%，而本股跌了 Y%（Y > X），则 X% 中的一部分可归因于ETF联动
- ETF 贡献 ≈ 同行中位数跌幅 × (1 - 个股特异性调整)

### 因素 C：叙事恐慌

由媒体叙事和市场情绪驱动的抛售，与实际业务影响不成比例。

**识别方法**：
- 叙事偏差评分 ≥ 6.0/10（参考 narrative-validation skill）
- 事件与公司核心业务技术关联度低
- 叙事转向速度 < 48h

**估算方法**：
```
叙事恐慌贡献 = 实际跌幅 - Beta放大 - ETF联动 - 基本面重定价
```
（通常作为残差项计算）

### 因素 D：基本面重定价

市场对公司未来盈利能力、增长前景进行的合理重新评估。

**识别方法**：
- 公司管理层确认业务影响
- 分析师下调盈利预期或目标价
- 实际财务数据出现恶化

**估算方法**：
```
基本面重定价 = 过度反应倍数中"合理市值调整"部分 / 总市值 × 100%
```

## 第二步：元模式匹配

根据四因素分解结果，逐一检查以下元模式：

### 模式 1: 关键词驱动恐慌
**触发条件**（需全部满足）：
- 叙事偏差评分 ≥ 8.0/10
- 事件与公司业务技术关联度 ≤ 2/10
- 叙事转向时间 < 48h
- 基本面重定价因素占比 < 10%

**含义**：市场对关键词组合的条件反射式抛售，恢复概率高（>80%），时间快（3-6个月）。

### 模式 2: 行业误分类
**触发条件**：
- 公司实际受威胁业务线占总收入 < 40%
- 但被市场/ETF归入100%受威胁的行业分类
- 同行中真正受威胁的公司跌幅反而可能更小

**含义**：ETF和被动策略的机械性卖出，需业务分拆重分类。

### 模式 3: 高Beta放大
**触发条件**：
- Beta > 1.5
- Beta放大贡献占总跌幅 > 30%

**含义**：跌幅中相当比例是股票属性而非业务风险，市场情绪恢复后自动回收。

### 模式 4: 期权-股票背离
**触发条件**：
- 股票层面 SEVERE/EXTREME
- 期权市场聪明钱方向为"偏多"或"强烈看多"

**含义**：机构投资者不认同恐慌定价，历史上此类背离通常在1-3个月内收敛。

### 模式 5: 技术面-基本面背离
**触发条件**：
- 技术面：完全空头排列（价格 < 所有关键均线）
- 基本面：营收增速 > 25% 且 FCF 为正 且 毛利率 > 60%

**含义**：最强的V型反弹信号之一，下一次强劲财报通常是反转催化剂。

```
抛售归因分解：

| ETF/被动卖出 | XX% | -X.X% | 同行联动 |
| 叙事恐慌 | XX% | -X.X% | 叙事偏差X/10 |
| 基本面重定价 | XX% | -X.X% | 合理调整部分 |

匹配的元模式：
- [x/空] 关键词驱动恐慌
- [x/空] 行业误分类
- [x/空] 高Beta放大
- [x/空] 期权-股票背离

```
